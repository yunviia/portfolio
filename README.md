# 👋 안녕하세요!

저는 AI·IoT 전공을 기반으로 웹 프론트엔드와 AI 서비스 개발을 공부하고 있는 개발자 지망생입니다.  
단순히 작동하는 코드에 그치지 않고, 사용자 경험과 완성도를 고려한 서비스를 만드는 것을 목표로 하고 있습니다.  

## 🌱 현재 학습 중
- Python, HTML/CSS, JavaScript  
- React 및 프론트엔드 프레임워크  
- 데이터 처리 및 시각화  
- API 활용 프로젝트  

## 💡 관심 분야
- 웹 프론트엔드 개발  
- AI·데이터 기반 서비스  


## 🎯 목표
대외활동과 팀 프로젝트를 통해  
기획 → 개발 → 배포 전 과정을 경험하고,  
실무형 역량을 갖춘 개발자로 성장하는 것입니다.  

📌 이 깃허브는 앞으로 학습 과정과 프로젝트 결과물을 기록하는 공간입니다.
